# 上传目录

这个目录用于存放准备上传的资料。

## 📁 目录结构建议

你可以按课程组织文件（可选）：

```
upload/
├── 数据结构/
│   ├── 2024-期末-试卷.pdf
│   └── 2024-期末-答案.pdf
├── 线性代数/
│   └── 2024-复习-总结.pdf
└── 其他文件.pdf
```

或者直接放在根目录：

```
upload/
├── 数据结构-期末试卷.pdf
├── 线性代数-复习资料.pdf
└── ...
```

## 🚀 使用方法

1. **将资料放入此目录**
   - 支持任意目录结构
   - 文件名最好包含课程、年份、类型等信息

2. **运行脚本**
   ```bash
   cd /workspaces/FDU-Sharing
   python scripts/easy_pr.py
   ```

3. **按提示操作**
   - 选择要处理的文件
   - 确认课程信息
   - 选择资料类型
   - 确认文件名

4. **自动创建PR**
   - 脚本会自动创建Git分支
   - 移动文件到正确位置
   - 更新MDX页面
   - 创建Pull Request

## ⚠️ 注意事项

- 文件大小不超过 100MB
- 支持的格式：PDF, Word, PPT, ZIP, 图片等
- 文件名避免使用特殊字符
- 处理完成后可以删除此目录中的文件

## 💡 提示

脚本会根据文件名智能识别：
- **课程名**（如果在子目录中）
- **年份**（2024、2023等）
- **资料类型**（期末、期中、复习等）

建议文件名包含这些信息，方便自动识别！
